# UI/UXデザイン

本ドキュメントでは、ゲーム全体のUI/UXデザイン方針と各画面の仕様を定義します。

## 1. 概要

### デザイン方針

| 項目 | 内容 |
|:---|:---|
| **トーン** | Minecraftバニラ準拠 |
| **情報量** | 情報リッチ（利便性重視） |
| **サーバー間差異** | 生活/プレイグラウンドで表示内容を変更 |

### 設計思想

* Minecraftの雰囲気を壊さないデザイン
* サーバーの性質上、必要な情報は常に表示
* 場面に応じて最適な情報を提供

## 2. UI要素一覧

### 利用可能なUI要素

| UI要素 | 特徴 | 主な用途 |
|:---|:---|:---|
| **チャット** | テキスト表示、クリック可能 | システムメッセージ、通知 |
| **アクションバー** | ホットバー上のテキスト、常時表示可 | 戦闘ステータス |
| **タイトル/サブタイトル** | 画面中央の大きなテキスト | 重要な通知、イベント開始 |
| **ボスバー** | 画面上部のプログレスバー | ランク、目標表示 |
| **スコアボード** | 画面右側のリスト | ステータス、進捗表示 |
| **インベントリGUI** | チェストUIを活用したメニュー | ショップ、クエスト受注、設定 |

### 用途別割り当て

| 用途 | 生活サーバー | プレイグラウンド |
|:---|:---|:---|
| **ボスバー** | ランク・経験値・総合レベル | 目標バイオーム・方角・距離 |
| **スコアボード** | Δコイン・オンライン人数 | 探索者数・経過時間・移動距離・クエスト進捗 |
| **アクションバー** | 未使用 | 戦闘ステータス |

### ボスバー

| 要素 | 表示形式 | 説明 |
|:---|:---|:---|
| **目標バイオーム** | 目標: ジャングル | 救出バイオーム名 |
| **方角** | → 北東 | 目標への方角 |
| **距離** | 約650m | 目標までの概算距離 |

### スコアボード

| 行 | 表示形式 | 説明 |
|:---|:---|:---|
| **1** | 探索者: 8人 | プレイグラウンド内のプレイヤー数 |
| **2** | 経過: 05:32 | 侵入からの経過時間 |
| **3** | 移動: 423m | 累計移動距離 |
| **4** | Daily: 67% | デイリークエスト進捗（平均） |
| **5** | Quest: 45% | ノーマルクエスト進捗（平均） |

### アクションバー

常時表示される戦闘ステータス。

| 記号 | ステータス | 表示例 |
|:---|:---|:---|
| ⚔ | 攻撃力 | 12.5 |
| ☆ | 会心率 | 15% |
| ⚡ | 会心ダメージ | 32% |
| ♥ | HP（現在/最大） | 18/20 |
| 🛡 | 防御力 | 8 |

#### 表示形式

```text
⚔12.5 ☆15% ⚡32% ♥18/20 🛡8
```

## 5. レベル・ランクシステム

### ランク

| 項目 | 内容 |
|:---|:---|
| **上昇条件** | デイリー任務、他プレイヤーとの交流、プレイ時間 |
| **役割** | プレイ量の可視化（ゲーム進行には直接関係なし） |
| **表示** | ★ + 数字（例: ★5） |

### 3要素レベル

| レベル | 上昇条件 | 説明 |
|:---|:---|:---|
| **戦闘レベル** | 与ダメージ、受ダメージ、武器種等 | 戦闘経験の指標 |
| **建築レベル** | 自分の区画でのブロック設置・破壊量 | 建築活動の指標 |
| **生活レベル** | プレイ時間、アイテム作成・購入・使用 | 生活活動の指標 |

### 総合レベル

* **計算式**: 戦闘レベル + 建築レベル + 生活レベル
* **表示形式**: Lv.XXX（例: Lv.95）

## 6. ステータスシステム

### 基本ステータス

| ステータス | 記号 | 基準値 | 説明 |
|:---|:---|:---|:---|
| **HP** | ♥ | 20 | バニラ基準、装備等で変動 |
| **防御力** | 🛡 | 0 | 独自ステータス、ダメージ軽減 |
| **攻撃力** | ⚔ | 1.0 | 基礎ダメージ倍率 |
| **会心率** | ☆ | 0% | クリティカル発生確率 |
| **会心ダメージ** | ⚡ | 50% | クリティカル時の追加ダメージ |

### ステータス変動要因

| 要因 | 影響するステータス |
|:---|:---|
| **武器** | 攻撃力、会心率、会心ダメージ |
| **防具** | HP、防御力 |
| **アクセサリ** | 各種ステータス |
| **消耗品** | 一時的なバフ/デバフ |

## 7. 通知システム

### 通知種別

| 種別 | UI要素 | 用途 |
|:---|:---|:---|
| **重要通知** | タイトル/サブタイトル | イベント開始、クエスト完了 |
| **一般通知** | チャット | システムメッセージ、ヒント |
| **緊急通知** | 画面効果 + 音 | 低HP警告、救出可能通知 |

### タイトル使用例

| シーン | タイトル | サブタイトル |
|:---|:---|:---|
| **プレイグラウンド侵入** | プレイグラウンド | 生存せよ |
| **クエスト完了** | クエスト完了！ | +150 Δコイン |
| **救出可能** | 救出準備完了 | 目標バイオームで夜を待て |
| **帰還成功** | 帰還成功 | お疲れ様でした |

## 8. カラースキーム

### 基本カラー

Minecraftバニラのカラーコードを基準とする。

| 用途 | カラー | コード |
|:---|:---|:---|
| **通常テキスト** | 白 | `§f` |
| **強調** | 黄 | `§e` |
| **成功/正常** | 緑 | `§a` |
| **警告** | 金 | `§6` |
| **エラー/危険** | 赤 | `§c` |
| **情報** | 水色 | `§b` |
| **レア/特殊** | 紫 | `§d` |

### 用途別カラー

| 要素 | カラー | 理由 |
|:---|:---|:---|
| **Δコイン** | 金（§6） | 通貨を示す |
| **HP** | 赤（§c） | 体力を示す |
| **ランク** | 黄（§e） | 重要な指標 |
| **レベル** | 緑（§a） | 成長を示す |
| **クエスト進捗** | 水色（§b） | 情報を示す |
