# アイテムデータベース：インデックス (Item Database Index)

## 概要
本プロジェクトで使用される全てのカスタムアイテムを管理するデータベースの入り口です。
各カテゴリの詳細、CMD（CustomModelData）の割り当て、命名規則を定義します。

| **9000 - 9999** | **その他** | - |

---

## 2. 共通NBTタグ規約

システムでアイテムを識別するため、以下のNBTタグを付与します。

| NBTキー | 値の例 | 説明 |
|---------|--------|-----|
| `TechItem` | 1 | 本プロジェクトのカスタムアイテムである識別フラグ |
| `ItemCategory` | "melee" | カテゴリ識別用 |
| `ItemTier` | 1 | ティア（ランク） |
| `WeaponID` | "shi-01" | 固有ID |
