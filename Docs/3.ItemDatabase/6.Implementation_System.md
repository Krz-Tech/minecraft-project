# アイテム実装技術仕様 (Item Implementation System)

## 概要
Tech::Playgroundにおけるアイテムの実装・変換ロジック、およびNBT設計について定義します。

## 1. 開発スタック
*   **CustomCrafting (CC)**: バニラレシピの「[強化]版への変換」およびカスタムレシピ管理。
*   **ExecutableItems (EI)**: 高性能装備（ID付与アイテム）の独自ロジック、スキル、属性の実装。

## 2. バニラ武器の強化変換仕様
バニラの標準レシピで作成されたアイテムを、Tech仕様に変換します。

*   **アイテム名**: `元の名称 [強化]` (例: `鉄の剣 [強化]`)
*   **Custom Model Data**: サイバーパンク風の外見を適用。

### 変換ロジック (Skript/CustomCrafting)
CustomCraftingの成果物に以下のNBTを付与します。補助的にSkriptで以下の処理を行う場合があります。

```skript
on craft:
    if event-item is iron sword or diamond sword:
        set {_item} to event-item
        set name of {_item} to "%name of {_item}% [強化]"
        set nbt of {_item} to nbt compound from "{CustomModelData:2001, TechItem:1}"
        set event-item to {_item}
```

## 3. 技術的共通設計
*   **識別タグ**: 全てのカスタムアイテムには `TechItem: 1` タグを付与。
*   **耐久値表示**: `Unbreakable: 1` を設定し、アクションバーやLoreで独自の耐久値を表示。
*   **属性管理**: 衝撃力 (Impact) や重量 (Weight) はNBTで管理し、Skriptで計算。
