# スコアボード・サイドバー表示仕様 (Scoreboard & Sidebar Specifications)

## 概要
プレイヤーの画面右側に表示されるスコアボード（サイドバー）の表示仕様です。
「テックスーツの内蔵バイザーに投影されるHUDデータ」という設定に基づき、状況に応じた情報を整理して提供します。

---

## 1. 基本ルール
*   **タイトル**: 16文字以内。ボールドやカラーを使用して視認性を高める。
*   **更新間隔**: 
    *   通貨、レベルなどはイベント発生時に即時更新。
    *   距離、時間は1秒間隔で更新。
*   **非表示設定**: 設定GUIより、特定の情報を非表示にできるオプションを検討。

---

## 2. ロビー表示 (Safe World Mode)
生活ワールド、ロビー等の安全地帯での表示です。

### 構成案
| 行数 | 表示内容 | Skript変数例 (参考) | 備考 |
| :-- | :--- | :--- | :--- |
| Title | `&b&lTECH::PLAYGROUND` | - | |
| 1 | `&8&m------------------` | - | 区切り線 |
| 2 | `&7User: &f<Name>` | `%player%` | |
| 3 | `&7Rank: &b<Rank>` | `{rank::%uuid%}` | F1〜SSS |
| 4 | `&7Lv: &3C-%{c_lv}% / L-%{l_lv}%` | `{level::combat::%uuid%}`, `{level::life::%uuid%}` | C:戦闘 / L:生活 |
| 5 | `&r` | - | 空行 |
| 6 | `&7Balance: &6%{money}% Δ` | `{balance::%uuid%}` | |
| 7 | `&7Storage: &f%{st_u}%/%{st_m}%` | `{storage::used::%uuid%}`, `{storage::max::%uuid%}` | 倉庫使用率 |
| 8 | `&r` | - | 空行 |
| 9 | `&7Loc: &f%{district_name}%` | `{current_district::%uuid%}` | 地区名 |
| 10 | `&7Network: &bOnline` | - | サーバー状態 |
| 11 | `&7Status: &aCONNECTED` | - | 常時表示 |

## 3. プレイグラウンド表示 (Incursion Mode)
プレイグラウンド（戦闘/探索エリア）での表示です。

### 構成案
| 行数 | 表示内容 | Skript変数例 (参考) | 備考 |
| :-- | :--- | :--- | :--- |
| Title | `&c&lINCURSION ACTIVE` | - | 赤色で警告感を演出 |
| 1 | `&8&m------------------` | - | |
| 2 | `&7Danger: &cClass-%{d_lv}%` | `{pg::danger_level}` | B, A, Sなど |
| 3 | `&7Integrity: &aSTABLE` | `{pg::suit_integrity::%uuid%}` | スーツの整合性 |
| 4 | `&r` | - | 空行 |
| 5 | `&7Est.Loot: &6+%{session_delta}% Δ` | `{pg::session::loot::%uuid%}` | 推定獲得金額 |
| 6 | `&7Defeated: &c%{session_kills}%` | `{pg::session::kills::%uuid%}` | 脅威排除数 |
| 7 | `&r` | - | 空行 |
| 8 | `&7Vanish: &b%{dist_ext}%m` | `{pg::dist_to_exit::%uuid%}` | 帰還地点までの距離 |
| 9 | `&7Limit: &f%{session_time}%` | `{pg::time_limit::%uuid%}` | 帰還までの制限時間 |
| 10 | `&r` | - | 空行 |

---

## 4. 視認性に関するガイドライン
*   **多重更新の回避**: スコアボードの更新はサーバー負荷に直結するため、数値を直接表示するのではなく、0.5秒〜1秒程度のバッファを持たせて更新を行う。
*   **色の意味付け**:
    *   `&b` (Cyan): システム正常、ナビゲーション。
    *   `&c` (Red): 危険、警告、戦闘。
    *   `&6` (Gold): 経済、重要アイテム。
    *   `&a` (Green): 成功、回復、安定。

## 5. 検討事項 (今後の拡張)
*   **パーティー表示**: パーティーを組んでいる場合、メンバーのHP/AP状態を数行表示するべきか。
*   **ミッション表示**: アクティブなミッションの進捗を表示するトグル機能。
