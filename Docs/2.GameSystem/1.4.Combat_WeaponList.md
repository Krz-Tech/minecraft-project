# 戦闘システム：武器カテゴリと入手経路 (Weapon Categories & Acquisition)

## 概要
Tech::Playgroundで使用される主要な武器カテゴリと、プレイヤーがそれらを入手する方法について定義します。
具体的なアイテムデータについては [アイテムデータベース (Docs/3.ItemDatabase/)](../3.ItemDatabase/0.ItemDatabase_Index.md) を参照してください。

## 1. 武器の種類
本プロジェクトでは、近接・遠距離の枠を超えた「Tech武器」を展開します。

*   **強化バニラ武器**: 鉄やダイヤの武器を強化・変換したもの。[実装詳細はこちら](../3.ItemDatabase/6.Implementation_System.md)
*   **EN（エナジー）武器**: 高い衝撃力と視覚効果を持つ上位装備。
*   **重火器**: プレイグラウンドでのみ性能を発揮するSF兵器。

## 2. 入手経路 (Acquisition)

1.  **クラフト**: 生活ワールドでバニラ素材から「[強化]」版を作成。
2.  **ショップ購入**: 完成済みの「Krzシリーズ」や「EN武器」をΔコインで購入。
3.  **プレイグラウンド (PG)**: 敵やチェストから現品（または設計図）をドロップ。

## 3. 装備制限
*   **Tech Suit依存**: 一部の高度な武器は、特定の出力を備えた Tech Suit を着用していないと使用できません。
*   **ランク制限**: 高性能な武器はプレイヤーのランク (F〜SSS) によって使用・購入が制限されます。
