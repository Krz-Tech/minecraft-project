# 通貨システム

本ドキュメントでは、ゲーム内通貨「Δコイン」の仕様と経済バランスを定義します。

## 1. 概要

### 基本情報

| 項目 | 内容 |
|:---|:---|
| **通貨名** | Δコイン（デルタコイン） |
| **単位** | Δ |
| **用途** | アイテム購入、サービス利用 |
| **入手方法** | プレイグラウンド探索、クエスト報酬 |

### 経済設計思想

| 方針 | 内容 |
|:---|:---|
| **建材は激安** | 建築を促進するため大量生産品として安価に設定 |
| **装備は高価** | やりくりが必要なバランス |
| **探索が基本** | ショップ購入より探索で素材収集が効率的 |

## 2. 生活サーバーの制約

### クラフト制限

| 項目 | 状態 |
|:---|:---|
| **インベントリクラフト** | 封印 |
| **作業台クラフト** | 封印 |
| **専用クラフト台** | 独自素材・ツールのみクラフト可 |
| **バニラアイテム入手** | 購入のみ |

### 世界観的理由

* 建材は加工技術により大量生産されており安価
* 素材（鉱石等）は採掘・栽培が必要なため高価
* バニラアイテムのクラフトは技術的に封印されている

## 3. ショップ構成

### 恒常ショップ

常時利用可能なショップ。バニラアイテムのみ販売。

| カテゴリ | 販売品 |
|:---|:---|
| **建材** | 石、木材、ガラス、レンガ等 |
| **食料** | パン、ステーキ、金リンゴ等 |
| **武器** | 木〜ネザライト武器 |
| **防具** | 木〜ネザライト防具 |
| **ツール** | 木〜ネザライトツール |

### 時限ショップ

不定期に出現する特別ショップ。

| 項目 | 内容 |
|:---|:---|
| **販売品** | 独自素材（稀） |
| **価格** | 高価格 |
| **設計意図** | 探索の方が効率的になるよう設定 |

## 7. ゲーム進行と経済

### 想定進行フロー

```text
[初期]
  │ 所持: 0Δ
  │ 装備: なし
  ↓
[序盤] 探索2〜3回
  │ 所持: 200〜500Δ
  │ 購入: 石装備 → 鉄装備
  ↓
[中盤] 探索15〜25回
  │ 所持: 2,000〜4,000Δ
  │ 購入: ダイヤ装備
  │ 開始: 独自素材収集
  ↓
[終盤] 探索50回〜
  │ 独自素材蓄積
  │ クラフト: 独自装備
  │ ネザライト装備は不要に
  ↓
[エンドゲーム]
    独自装備の強化・収集
    建築・生活コンテンツへ
```

### 装備入手タイミング

| 装備 | 必要Δコイン | 必要探索回数 | 備考 |
|:---|:---:|:---:|:---|
| 石装備一式 | 約100Δ | 1回 | 初期装備 |
| 鉄装備一式 | 約750Δ | 4〜5回 | 序盤の目標 |
| ダイヤ装備一式 | 約3,900Δ | 20〜25回 | 中盤の目標 |
| ネザライト装備一式 | 約19,400Δ | 100回以上 | 購入時には不要 |
| 独自装備（低級） | クラフト | 素材収集 | 中盤から |
| 独自装備（高級） | クラフト | 素材収集 | 終盤 |

## 8. 修繕システム

### 概要

装備には耐久値が存在し、使用により消耗する。修繕は村人に依頼して行う。

### 修繕コスト

| 必要なもの | 内容 |
|:---|:---|
| **対応素材** | 装備のティアに応じた素材 |
| **Δコイン** | 修繕手数料 |

### 修繕費用（案）

| ティア | 必要素材 | 手数料 |
|:---|:---|:---:|
| **鉄装備** | 鉄インゴット×1〜2 | 10Δ |
| **ダイヤ装備** | ダイヤモンド×1 | 50Δ |
| **ネザライト装備** | ネザライトインゴット×1 | 200Δ |
| **独自装備** | 対応独自素材 | 100〜500Δ |

### 設計意図

* 装備の維持にコストがかかることで経済が循環
* プレイグラウンド探索の継続的な動機付け
* 素材の消費先を確保
