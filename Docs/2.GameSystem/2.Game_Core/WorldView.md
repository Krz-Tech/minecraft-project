# 世界観設定

本ドキュメントでは、プロジェクトの世界観・設定・歴史を定義します。Minecraftの基本世界観を踏襲しつつ、独自の解釈と設定を加えた世界です。

## 1. 世界の概要

### 基本設定

| 項目 | 内容 |
|:---|:---|
| **時代** | 「The Peaceful Update」から数年後 |
| **世界の状態** | 全ディメンションのモブが友好化した平和な世界 |
| **例外** | 古代都市（ディープダーク）、プレイグラウンド |
| **通貨** | Δコイン（デルタコイン） |

### 世界の特徴

* オーバーワールド、ネザー、エンドの全ディメンションでモブが友好化
* 各ディメンション間で貿易が行われ、モブ同士の経済が成立
* プレイヤー（クラフター）は探索者・冒険者として活動

## 2. 歴史

### 2人の伝説的プレイヤー

かつて、2人の伝説的なプレイヤーが存在した。彼らは友人としてこの世界に降り立ったが、その性格は対照的だった。

| プレイヤー | 性格 | 功績 |
|:---|:---|:---|
| **1人目** | 優しさ・調和を重んじる | The Peaceful Updateを成し遂げ、世界を平和に導いた |
| **2人目** | スリル・冒険を求める | Delta Guildの前身を築き、探索の道を開いた |

* 誰も彼らの姿を見たことがない
* 名前すら記録に残っていない
* しかし、何を成し遂げたかは記録として残り、世界中が知っている

### The Peaceful Update

1人目のプレイヤーは武力ではなく「優しさ」によって全てのモブを救い、敵対していた存在たちと和解を果たした。

この出来事は「**The Peaceful Update**」と呼ばれ、以降の世界は平和な時代を迎えることとなる。

### 探索者の礎

2人目のプレイヤーは、平和な世界においても冒険とスリルを求め続けた。彼はエンドの研究者たちと協力し、未知の領域への扉を開く研究を進めていたとされる。この研究と組織が、後のDelta Guildへと発展した。

### 失踪とプレイグラウンドの出現

ある日、2人のプレイヤーは突然姿を消した。

そして、彼らが消えたのと同じタイミングで「プレイグラウンド」が出現した。バグまみれのスナップショットが次元の狭間に現れたのだ。

* 2人の失踪とプレイグラウンドの出現には何らかの関係があると推測されている
* 各ディメンションの研究機関が長年調査を続けているが、真相には至っていない
* 彼らがどこへ消えたのか、なぜプレイグラウンドが現れたのか—それは今も最大の謎である

### 現在

The Peaceful Updateから数年が経過。世界は平和を享受している。

新たにこの世界にやってきた「クラフター」たちは、Delta Guildに所属しプレイグラウンドの探索を行う。彼らは新参者であり、世界の歴史や伝説について何も知らない状態からスタートする。

> **プレイヤー視点**: クラフターとして世界に降り立った時点では、The Peaceful Updateや伝説のプレイヤーについて知らない。ギルドでの活動やモブたちとの交流を通じて、徐々に世界の歴史を知ることになる。

## 3. ディメンション

### オーバーワールド

* 全モブが友好化した平和な世界
* クラフターたちの主な生活拠点
* 古代都市のみが例外的な危険地帯として残存

### ネザー

* 友好化されたネザーモブが居住
* オーバーワールド・エンドとの貿易が活発
* ピグリンとの交易が盛ん

### エンド

* エンダードラゴンとも和解済み
* エンダーマンたちが独自の文化を形成
* 高価値な資源の産地として知られる

### プレイグラウンド

詳細は [Playground.md](../2.Game_Battle/Playground.md) を参照。

* オーバーワールド・ネザー・エンドの中間に位置する異質な次元
* 「忘れ去られたスナップショット」—バグまみれのまま放棄された空間
* 敵対モブが依然として存在する唯一の広大な領域
* 旧バージョンのMinecraftを彷彿とさせるビジュアル

## 4. 古代都市（Ancient City）

### 概要

ディープダークに存在する古代都市は、The Peaceful Updateの影響を受けなかった唯一のオーバーワールド内危険地帯。

| 項目 | 内容 |
|:---|:---|
| **位置** | オーバーワールド中心都市の地下（ディープダーク） |
| **状態** | 敵対モブ（ウォーデン）が存在 |
| **現在の管理** | Delta Guildによる制圧・管理 |
| **重要施設** | プレイグラウンドポータル |

### 古代都市とギルド本部の関係

古代都市には失われた高度な技術が眠っていた。Delta Guildの前身となる研究チームは、この技術を解析・開発するために古代都市の直上に研究施設を建設。これが現在のギルド本部である。

やがてギルド本部を中心に人やモブが集まり、自然発生的にオーバーワールドの中心都市が形成された。

```
【地理的構造】
地上: オーバーワールド中心都市
  └─ Delta Guild本部
      ↓
地下: 古代都市（ディープダーク）
  └─ プレイグラウンドポータル
```

### プレイグラウンドポータル

* 古代都市の技術を使用して建造
* 古代都市内に設置されている
* 不安定な技術のため、テレポート先の座標は制御不能
* ポータルは片道—プレイグラウンド側にポータルは出現しない
* ギルド本部から地下へ降りることでアクセス可能

## 5. クラフター（プレイヤー）

### 定義

「クラフター」とは、この世界に新しく参加したプレイヤーの総称。伝説のプレイヤーとは別の存在であり、平和な世界で新たな冒険を求める者たち。

### 特性

* 探検・スリル・作業を好む
* プレイグラウンドでの探索を主な活動とする
* 獲得したアイテムを売却し、Δコインを稼ぐ

### ランクシステム

クラフターの練度はランクによって可視化される。ランクはMinecraftの鉱物に基づく。

| ランク | 鉱物 |
|:---|:---|
| **最低** | 銅 |
| ↓ | 鉄 |
| ↓ | 金 |
| ↓ | ダイヤモンド |
| **最高** | ネザライト |

> **注意**: プレイグラウンドで死亡すると、ランク経験値の一部を失う。

## 6. 経済

### Δコイン（デルタコイン）

* 世界共通の通貨
* プレイグラウンドで獲得したアイテムの売却が主な収入源
* 詳細は [Currency.md](../2.Game_Economy/Currency.md) を参照

### 貿易

* ディメンション間でモブ同士の貿易が成立
* クラフターも貿易に参加可能
* プレイグラウンド産アイテムは希少価値が高い
