# Tech::Playground
## Build > Launch > Survive

> 「技術(コード)が創り出す、かつてない没入体験。」

## コンセプト
**Tech::Playground** は、Krz-Techが提供する次世代のMinecraftサーバープロジェクトです。
「DX（開発者体験）の最適化」と「UX（ユーザー体験）の最適化」を軸に、技術的な挑戦と純粋なゲームの楽しさを融合させた実験場（Playground）です。

## コアゲームループ
1. **Build (生活)**: 安全なサンクチュアリで拠点を築き、装備を整える。
2. **Launch (出撃)**: プレイグラウンド（危険地帯）へ出撃し、資源を確保する。
3. **Survive (生存)**: 敵や他プレイヤーの脅威から生き残り、戦利品を持ち帰る。

## キーワード
* **Namespace Style**: `Tech::Playground` - 技術と遊びの境界。
* **Tech Stack**: Cloudflare, Kubernetes, Skript, AI Agents.
* **Game Style**: Hybrid (Lifestyle MMO + Extraction Shooter).
